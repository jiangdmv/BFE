### Solution

https://developer.mozilla.org/en-US/docs/Web/CSS/WebKit_Extensions

```css
.max-three-lines {
  /* your code here */
  display: -webkit-box;
  overflow: hidden;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 3;
}
```
